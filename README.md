# Pincode Lookup Application

## Overview

This project is built using AppSheet, Google Sheets, and Google Apps Script.

The application allows users to enter an Indian pincode and automatically fetches the District, State, and Region using the India Post API.

## Technologies Used

- AppSheet
- Google Sheets
- Google Apps Script
- India Post API

## Workflow

1. User enters a pincode in the AppSheet form.
2. The pincode is stored in the **Pincode Input** sheet.
3. Google Apps Script calls the India Post API.
4. The fetched District, State, and Region are stored in the **Pincode Details** sheet.
