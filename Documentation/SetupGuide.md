# Setup Guide

## Prerequisites
- Google Account
- Google Sheets
- AppSheet

## Steps

1. Create a Google Sheet.
2. Create three sheets:
   - Users
   - Pincode Input
   - Pincode Details
3. Connect the Google Sheet to AppSheet.
4. Create an **Enter Pincode** form.
5. Open **Extensions → Apps Script**.
6. Paste the Apps Script code.
7. Save and authorize the script.
8. Test by entering a pincode in the app.
9. Verify that the **Pincode Details** sheet is updated automatically.
